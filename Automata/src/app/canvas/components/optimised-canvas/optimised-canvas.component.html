Using Canvas - Optimised with no Model

  • In this case we continue to generate a canvas via Typescript.
  • No Model - direct from loop to render
  • Still used class inheritance.
  • Chrome: 1920*1200 will give us a delay of ~4 seconds on production and a total memory usage of 47MB - Obvisouly the grahics card is taking care of this.
  • Chrome:: Rendering 4K resolution on dev of 3840 x 2160 will take around 15 seconds. One could optimise by not rendering the white space. Going from HD to 4K only adds another 0.5 MB of memory.
  • At this point there is no model and optimisations would center around parallel and/or buffered rendering and using web workers.
